## Twelve Thrones Genesis (The Epistemology Engine)

### Core Concept
> "We do not ask AI for truth. We make AI prove it — then seal it forever."

Query **12 frontier AI models in parallel**, detect their disagreement severity, weight responses by reliability, and **map the epistemic frontier** (where AI understands vs. where it's uncertain).

### Architecture

```
Question
  ↓
12 Frontier Models (Parallel Queries)
├─ Claude (Anthropic) — 0.98 weight
├─ GPT-4o (OpenAI) — 0.96 weight
├─ DeepSeek-R1 & V3 — 0.88 weight
├─ Grok (XAI) — 0.85 weight
├─ Gemini (Google) — 0.87 weight
├─ Llama 70B (Groq) — 0.82 weight
├─ Mistral — 0.80 weight
└─ [5 more specialized models]
  ↓
Weighted Consensus (0-100%)
  ↓
Disagreement Severity Analysis
(unanimous / strong / moderate / severe)
  ↓
Epistemic Mapping
├─ Agreement Zones (what models converge on)
├─ Disagreement Zones (where they fracture)
└─ Knowledge Frontiers (what would resolve uncertainty)
  ↓
Arweave Archive (permanent, immutable)
  ↓
Sui Blockchain (on-chain ledger)
  ↓
NFT Mint (disagreement made collectible)
```

### Key Mechanics

1. **Weighted Consensus**
   - Each model has a historical reliability weight
   - Claude's YES counts more than Llama's NO
   - Final score: weighted % (e.g., 69.4% YES)

2. **Disagreement Detection**
   - **Unanimous** (90%+): All models agree
   - **Strong** (70-89%): Clear majority
   - **Moderate** (50-69%): Split decision (knowledge frontier)
   - **Severe** (<50%): Fundamental disagreement

3. **Epistemic Mapping**
   - Extracts what models agree on (agreement zones)
   - Identifies contradictions (disagreement zones)
   - Defines testable hypotheses (knowledge frontiers)

4. **Permanent Archival**
   - Results stored on **Arweave** (~$0.04 per query)
   - On-chain ledger on **Sui** (~$0.03 per query)
   - Epistemic NFT minted as collectible

### Use Cases
- Corporate decision-making (understand epistemic landscape before committing resources)
- Research frontiers (identify where new research is needed)
- Public infrastructure (archive consensus events historically)
- Truth-seeking communities (collectively funded consensus rituals)

### Philosophy
> Most AI systems hide uncertainty behind confident answers. This system **reveals it**.

## Key Differences

### 1. **Scope of Reasoning**

**Twelve Thrones:**
- External ensemble (query 12 independent models)
- Models don't interact with each other
- Consensus via weighted voting
- Focus: detecting disagreement

**Ọmọ Kọ́dà:**
- Internal parliament (11 lobes are one organism)
- Lobes interact (ritual protocol)
- Consensus via ceremony (veto, silence, harmony)
- Focus: unified reasoning with internal transparency

### 2. **Memory Architecture**

**Twelve Thrones:**
- Permanent archive (Arweave NFTs)
- No historical context per query
- Each query is independent
- Blockchain records consensus event

**Ọmọ Kọ́dà:**
- Causal lineage (parent_hash chain)
- Each breath builds on previous ones
- Memory is walked (traversed backward)
- Each lobe reads different branches
- Blockchain maintains evolution state

### 3. **Privacy**

**Twelve Thrones:**
- No privacy (queries sent to external APIs)
- All reasoning visible to model providers
- Consensus happens publicly

**Ọmọ Kọ́dà:**
- Private reasoning in TEE (Nautilus SGX)
- Public proofs on blockchain
- Memory encrypted (Walrus)
- Verifiable but private

### 4. **Embodiment**

**Twelve Thrones:**
- No embodiment (UI/API only)
- Results displayed to humans
- Human acts on insights

**Ọmọ Kọ́dà:**
- Physical body (Unitree G1 robot)
- Emotion-driven motion
- Autonomous execution
- Sensory feedback loop

### 5. **Reasoning Process**

**Twelve Thrones:**
```
Question → 12 Models (parallel) → Weighted Voting → Consensus
```

**Ọmọ Kọ́dà:**
```
Breath → Orunmila (checks lineage) → Lobes 1-11 (cascade)
  → Veto Sweep → Silence Check → Contradiction Detection
  → Harmonization → Execution → Àṣẹ Seal
```

### 6. **Scale of Intelligence**

**Twelve Thrones:**
- **Intelligence source**: 12 external models
- **Reasoning horizon**: Single query
- **Knowledge**: What models trained on

**Ọmọ Kọ́dà:**
- **Intelligence source**: Internal reasoning (11 lobes) + LLM inference
- **Reasoning horizon**: Lineage-aware (remembers past breaths)
- **Knowledge**: Learned + memorable

### 7. **Failure Modes**

**Twelve Thrones:**
- One API failure → graceful degradation (fewer models)
- Models can contradict each other → features, not bugs
- Consensus weight drops if models disagree

**Ọmọ Kọ́dà:**
- One lobe fails → ritual still proceeds (other 10)
- Veto stops everything immediately (safety mechanism)
- Silence quorum or depth threshold triggers Twelfth Face (graceful halt)

### 8. **Use of "12" / "Thrones"**

**Twelve Thrones:**
- **12** = the number of frontier AI models queried
- **Thrones** = metaphor for AI models as "seats of wisdom"
- **Genesis** = birth event when consensus is recorded

**Ọmọ Kọ́dà:**
- **11 lobes** = internal specialized perspectives
- **12th face** = emergence/silence (not counted like the 11)
- **Thrones** reference: Yoruba cosmology (Òrìṣà), not voting
- **Genesis** = organism's birth (Soul contract creation)

- Tools for uncertainty quantification
